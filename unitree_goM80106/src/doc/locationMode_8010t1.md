# locationMode_8010t1.cpp 解读

## 概述

该程序控制 Unitree GO-M8010-6 电机，实现**缓变位置闭环**功能。电机输出端目标位置为 5.0 rad，程序从当前实际位置出发，以固定步长逐步逼近目标，避免瞬间大力矩冲击。

## 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `gearRatio` | `-6.33 * queryGearRatio(GO_M8010_6)` | 减速比（负号表示输出端与电机轴转向相反） |
| `targetPos` | `5.0f * gearRatio` | 输出端目标位置 5.0 rad，换算到电机轴 |
| `stepSize` | `0.70f` | 每步最大步进量（电机轴 rad），控制逼近速度 |
| `kp` | 0.15 | 位置刚度系数 |
| `kd` | 0.10 | 阻尼系数 |

## 程序流程

### 1. 初始化阶段（L29-L39）

先发送一次空指令（`kp=0, kd=0, q=0`）读取电机当前实际位置 `data.q`，将 `cmdPos` 初始化为当前值。这样做防止启动时从 0 出发导致电机突然反转。

### 2. 主循环（L41-L68）

- **位置逼近逻辑**（L44-L47）：计算 `targetPos` 与 `cmdPos` 的差值，若差值大于 `stepSize`，则以 `stepSize` 为步长逼近一步；否则直接设定为目标值。
- **位置闭环控制**（L49-L58）：以 FOC 模式发送 `kp=0.15, kd=0.1, q=cmdPos`，驱动电机跟踪目标位置。
- **打印反馈数据**（L60-L65）：每周期输出电机当前角度、温度、转速、错误码。
- **周期 200μs**（L67）：控制频率约 5kHz。

### 3. 安全撤力阶段（L70-L88）

收到 `SIGINT` 信号（Ctrl+C）后退出循环，进入两步撤力流程：

**第 1 步 — 渐进撤力（L71-L78）**
- 执行 100 个周期（约 20ms）
- `kp` 从 0.15 线性递减至 0（位置刚度逐渐消失）
- `kd` 从 0.1 线性递增至 0.2（阻尼加大，吸收残余动能）
- 位置目标保持不变

**第 2 步 — 停止模式（L81-L87）**
- 切换 `mode = 0`（停止模式）
- 所有指令清零（`q=0, dq=0, tau=0, kp=0, kd=0`）
- 电机进入无力矩状态

## 设计要点

1. **缓变逼近**：每次循环最多变化 0.70 rad（电机轴），避免大阶跃导致过流或机械冲击。
2. **防突启动**：初始化时读取当前实际位置作为起点，而非从 0 开始。
3. **撤力安全**：Ctrl+C 后先逐步撤刚度再加阻尼，最后切停止模式，防止急停导致电机反弹或机构损伤。
4. **信号处理**：使用 `sig_atomic_t` 标志位响应 `SIGINT`，保证异步安全。
